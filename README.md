# 魔法肉鸽卡牌游戏 (DND 5.5E)

这是一款运行在 AstrBot 机器人框架上的文字策略肉鸽卡牌游戏，无需LLM。

## 游戏玩法

玩家将扮演法师领主，在由遭遇战、精英怪、随机事件、营地和商店组成的分支路线中前行。通过合理规划每回合的动作（A）与附赠动作（BA）打出法术、部署护符并召唤随从，指挥随从协同作战，在保持血量和收集卡牌的过程中不断变强，最终战胜关底的BOSS。

## 游戏指令

游戏内所有交互通过在会话中输入子指令完成：

| 指令 | 说明 | 示例 |
| :--- | :--- | :--- |
| `/rogue 开启` | 开启一局新游戏 | `/rogue 开启` |
| `/rogue 状态` | 查看玩家当前的血量、护盾、金币以及战场Buff | `/rogue 状态` |
| `/rogue 牌组` | 查看当前拥有的全部卡牌 | `/rogue 牌组` |
| `/rogue 使用 <序号> [目标]` | 消耗动作打出手牌。目标包括敌方（e1-eN）和我方（p0为自己，p1-p5为随从） | `/rogue 使用 1 e1` |
| `/rogue 随从 <我方格子> 攻击/技能 <目标>` | 控制我方格子的随从进行攻击或发动技能 | `/rogue 随从 1 攻击 1` |
| `/rogue 选择 <序号>` | 在地图分岔路口、随机事件抉择、商店购买时进行选择 | `/rogue 选择 1` |
| `/rogue 结束` | 结束本回合，进入敌方行动结算 | `/rogue 结束` |
| `/rogue 折叠` | 展开或折叠战斗及关卡界面的操作指南 | `/rogue 折叠` |
| `/rogue 放弃` | 放弃当前正在进行的战局并清空存档 | `/rogue 放弃` |

### 指令队列功能

支持一键批量发送指令：
- `/rogue 队列 [指令1, 指令2, ...]`（可简写为 `/rogue q` 或 `/rogue queue`）
- 示例：`/rogue 队列 [使用 1 e1, 随从 1 攻击 1, 结束]`
